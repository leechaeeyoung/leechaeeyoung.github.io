## ✔  파싱?
  다른 형식으로 저장된 데이터를 원하는 형식의 데이터로 변환하는 것

  - XML, JSON: 데이터 표현 문자열
  - JSON 파싱: JSON 형식 문자열을 객체로 변환하는 것
    - 동기: 시작해서 끝날때까지 한가지 작업만 실행
    - 비동기: 멀티스레딩이 가능
